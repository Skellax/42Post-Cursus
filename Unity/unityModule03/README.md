<h2> Description </h2>

This Module is the following of Module02, in this part you'll learn  to create a menu for your game, a pause menu and a UI 
interface to drop your turret. 

<h2>Exercice 00 A Simple Menu</h2>

<div align = "center">
  <img src= "https://github.com/Skellax/42Post-Cursus/blob/main/Unity/unityModule03/Menu.png" width=600/>
    </div>

  You need to create a menu scene. The button play send in map1 scene. You can update your menu  as and when
  tour project is approved, for exemple you can add a button select map to choose different map.

  <h2>Exercice 01: Drag and drop</h2>

  In this part you need to creat a UI menu for your game for drooping your turret. They have 3 type turret and 
  each turret have differents features dmg, rate for exemple.
  
  -First you need to add Drag and Drop system , it's not very difficult to understand (i give a  link in the end)
  
  -Second Time you need to create a energy reserve for your turrets. For my case i create a simple stock when i 
  placed my turret i discrease the count of energy system in relation to my turret. After 20seconds in game i 
  increase this energy.You need a alert message if your ennergy is to low when you try to place a turret when the cost is 
  higher to your energy.

  -Finally you can display different information for each turret , dmg, ammo , cooldown , you can create a animation
  whenyour turret is low (in my case i add a ammo bar above my turret)
